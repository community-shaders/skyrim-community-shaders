# Community Shaders Test Suite

Comprehensive test suite for the Skyrim Community Shaders project, focusing on validating changes in the current branch compared to the `dev` baseline.

## Overview

This test suite provides thorough coverage for:
- **C++ Code Changes**: Menu components, especially AdvancedSettingsRenderer and BackgroundBlur removal
- **HLSL Shader Refactoring**: New lighting system with LightingCommon.hlsli and LightingEval.hlsli
- **Hair Shader Improvements**: Refactored hair rendering with standardized structures
- **Code Structure Validation**: Ensuring architectural consistency and proper cleanup
